1. put(K key, V value) :	=> 전달된 키와 값 객체를 해당 Map에 추가시켜주는 메소드	
2. get(K key) :		=> 해당 Map으로 부터 전달된 키에 대한 값 객체를 반환해주는 메소드
3. set(K key, V value) :	=> 해당 Map에 전달된 키에 대한 값 객체를 전달된 값 객체로 변경해주는 메소드
4. keySet() :		=> 해당 Map에 저장된 키들만을 추출해서 Set 으로 담아 반환해주는 메소드
5. entrySet() :		=> 해당 Map에 저장된 키-값들을 추출해서 Set으로 담아 반환해주는 메소드

그 방법에 대해 설명하시오. (20점)
1. keySet() 이용하기
keySet()으로 Map의 key를 Set에 담은 후 Set에 있는 Iterator를 통해 접근
2. entrySet() 이용하기
entrySet()으로 Map의 키와 값(entry)을 Set에 담은 후 Set에 있는 Iterator를 통해 접근
