◆배열 
 - 같은 자료형을 모아놓은 공간(같은 의미)
	변수가 가정집이라면, 배열은 아파트로 비유할 수 있다.
 - 한 배열에는 같은 종류에 값을 저장하는게 좋다.
 - 반복문으로 효율적으로 사용★할 때 배열이 좋다.
 - 배열과 변수의 다른점
	기본 자료형으로 된 변수는 값을 저장
	배열은 주소를 저장
 - 배열 선언
	자료형 배열명[];
 - 배열 할당
	배열명[] = new 자료형[크기]; 	
 - 배열 선언과 동시에 할당
	자료형 배열명[] = new 바료형[크기];
 - 배열의 시작번지는 0번지 부터 크기 -1번지까지 접근 가능
 - 배열[번지] 	

 - 변수 복사



◆복사
 - 변수 복사
	int num1 =10, num2 =20;
	num1 = num2;
	num1 = 30;
	num2 = 20; //num1에 상관없이 20

- 배열 얕은 복사
	int arr1[] = new int[4];
	int arr2[];
	arr2 = arr1;
	arr2[0] = 4를 넣는다면 arr[0]도 4로 바뀜. 이는 배열은 주소값을 저장하고 복사또한 주소를 복사하므로 값이 같아짐.
 하나의 배열을 같이 공유하는 것
 - 배열 깊은 복사
 새로운 배열 객체를 생성하여 기존 배열의 데이터를 복사
