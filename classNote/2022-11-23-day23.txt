*배열
 - 같은 타입의 데이터들을 모아놓은 것
 - 크기 변경이 불가능
 - 배열 초기값
	- 정수형 변수 : 0
	- 실수형 변수 : 0.0, 0.0f
	- 참조 변수 : null

 - 배열 복사 방법
	- for문을 이용
	- System.arraycopy();	-> day06일차때 진행
	- Array.copy();
	- 참조 변수를 복사
		- 얕은 복사
			- 공유 => 다른 곳에서 수정하면 영향을 미친다.
		- 깊은 복사
			- 배열인 경우, 각 번지에 있는 값을 복사. 객체인 경우는 필드를 복사하는 경우 => 다른 곳에서 수정을 해도 영향이 없음!
 - 배열 초기화
	- 자료형 [] 배열명 = {값1, 값2, 값3};
 - 배열 사용법
	- 배열명 [번지]
